Please note that this is a first version of the HTML help files. There is not much content. If you want to contribute to the Frost project you can send us your HTML help pages, we will check the content and add them to the official help.

The HTML help files are contained in the help.zip file that is located in the help directory in your Frost directory. If you want to print some help pages, extract the help.zip file and open the HTML file in your browser.
